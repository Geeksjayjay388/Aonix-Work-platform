## ✨ Key Features

- **🚀 Centralized Dashboard**: Real-time project statistics, recent activity feed, and instant progress visualization.
- **👥 Client Management**: Securely store and manage client information with a streamlined interface.
- **📋 Advanced Project Tracking**: Comprehensive task management system for monitoring project milestones and deadlines.
- **💰 Financial Insights**: Dedicated payment tracking for invoices, revenue monitoring, and detailed financial records.
- **🎨 Premium Design System**: Sophisticated 8px-based spacing, professional slate-gray palette, and refined glassmorphism built on **Tailwind CSS 4**.
- **🔔 Interactive Feedback**: Instant toast notifications, smart form validation, and confirmation dialogs for destructive actions.
- **🌗 Theme Customization**: Native support for both Light and Dark modes, persisted across sessions.
- **📱 Fluid Responsiveness**: Optimized for desktop, tablet, and mobile devices with professional skeleton loaders for a smooth data experience.

## 🚀 Quick Start

### 1. Installation
Clone the repository and install the dependencies:
```bash
npm install
```

### 2. Environment Setup
Create a `.env` file in the root directory and add your Supabase credentials:
```env
VITE_SUPABASE_URL=your_project_url
VITE_SUPABASE_ANON_KEY=your_anon_key
```

### 3. Development
Start the local development server:
```bash
npm run dev
```

---

## 📖 Scripts

- `npm run dev`: Start the Vite development server.
- `npm run build`: Build the production bundle with TypeScript type checks.
- `npm run preview`: Serve the production build locally for testing.
- `npm run lint`: Run ESLint to ensure code quality and consistency.

## 💡 Pro Tips

- **Search Everything**: Use the search boxes in each module to quickly filter clients, projects, or tasks.
- **Keyboard Friendly**: Forms support full accessibility—use `Tab`, `Enter`, and `Esc` for a faster workflow.
- **Theme Sync**: Change your theme in Settings → Appearance; it will be saved automatically for your next visit.
- **Visual Feedback**: Watch for toast notifications in the bottom-right corner to confirm your actions.
