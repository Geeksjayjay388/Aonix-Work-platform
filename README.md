A sophisticated, professional-grade platform designed for creatives and businesses to streamline client relationships, project tracking, and financial management.

---

## Key Features

- **Centralized Dashboard**: Real-time project statistics, recent activity feed, and instant progress visualization.
- **Client Management**: Securely store and manage client information with ease.
- **Advanced Project Tracking**: Comprehensive task management system for monitoring project milestones.
- **Financial Insights**: Dedicated payment tracking for invoices, revenue monitoring, and financial records.
- **Premium Design System**: Sophisticated 8px-based spacing, professional slate-gray palette, and refined glassmorphism.
- **Accessibility & Performance**: WCAG 2.1 AA compliant colors, optimized for speed with Vite and React 19.

---

## Tech Stack

- **Frontend**: [React 19](https://react.dev/) with [TypeScript](https://www.typescriptlang.org/)
- **Build Tooling**: [Vite 8](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/) with a custom premium design system
- **Backend & Auth**: [Supabase](https://supabase.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

---

## Design Philosophy

The Aonix Platform emphasizes **Corporate Elegance** and **Usability**:
- **Professional Palette**: Muted Slate Gray (`#334155`) for a neutral, timeless aesthetic.
- **Clean Typography**: Replaces stylish fonts with **Inter** for maximum readability across all data densities.
- **Responsive Navigation**: Mobile-first sidebar and layout optimized for all screen sizes.
- **Subtle Interactions**: Snappy 200ms animations that provide natural feedback without distraction.

## Quick Start

### 1. Installation
Clone the repository and install dependencies:
```bash
npm install
```

### 2. Environment Setup
Create a `.env` file with your Supabase credentials:
```env
VITE_SUPABASE_URL=your_project_url
VITE_SUPABASE_ANON_KEY=your_anon_key
```

### 3. Development
Start the local development server:
```bash
npm run dev
```

---

## Scripts

- `npm run dev`: Start Vite development server.
- `npm run build`: Build production bundle with TypeScript checks.
- `npm run preview`: Serve production build locally for testing.
- `npm run lint`: Run ESLint to ensure code quality.
